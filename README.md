# WebViettel - Hệ thống Tra cứu và Đăng ký Gói cước Viettel

Hệ thống quản lý, tra cứu, so sánh và đăng ký các gói cước di động Viettel là nền tảng trực tuyến giúp người dùng dễ dàng tìm kiếm và lựa chọn gói cước tối ưu. Website tích hợp các bộ lọc thông minh, công cụ khảo sát nhu cầu cá nhân hóa và tính năng thanh toán ví điện tử giả lập. Hệ thống cũng hỗ trợ kết nối ví MetaMask để mang lại trải nghiệm thanh toán hiện đại trên mạng thử nghiệm Sepolia.

---

## Công nghệ sử dụng

- **Frontend**: React + TypeScript + TailwindCSS
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Web3 Integration**: MetaMask + Sepolia

---

## Chức năng chính

- **Tra cứu gói cước Viettel**: Tìm kiếm và lọc các gói cước phù hợp theo nhu cầu sử dụng.
- **Xem chi tiết gói cước**: Xem thông tin chi tiết về các ưu đãi data, gọi điện và giá cước.
- **Đăng ký gói cước**: Hỗ trợ khách hàng thực hiện đăng ký dịch vụ trực tuyến.
- **Quản lý gói cước đang sử dụng**: Kiểm tra gói cước hiện tại và theo dõi thời hạn sử dụng.
- **Theo dõi số dư và lịch sử giao dịch**: Nạp tiền ảo giả lập và lưu nhật ký giao dịch.
- **Hỗ trợ kết nối ví MetaMask**: Tích hợp ví tiền mã hóa hỗ trợ giao dịch Web3 thử nghiệm.
