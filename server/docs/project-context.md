Tech Stack:
Frontend:

- React
- TypeScript
- TailwindCSS
- TanStack Query
  Backend:
- Node.js
- Express
- MongoDB
- Mongoose
  AI:
- Google Gemini API
  Module đã hoàn thành:
- Authentication
- Package Listing
- Package Detail
- Subscription Engine
- Conflict Engine
- User Subscription
- Transaction History
- Notification System
  Yêu cầu:
  Không refactor các module đã ổn định. Không thay đổi schema hiện tại. Chỉ bổ sung tối thiểu để tích hợp chatbot.
