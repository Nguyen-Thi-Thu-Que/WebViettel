# Subscription Rules

Rule 1: Gói ngắn ngày: Được đăng ký nhiều lần.
Rule 2: Gói dài ngày: Không được đăng ký trùng.
Rule 3: Khác hệ: Được đăng ký song song.
Rule 4: Gói mua thêm (Addon): Được chạy song song với gói chính.
Rule 5: Một số gói yêu cầu Data nền.
Rule 6: Cùng hệ + cùng chu kỳ dài: Bị chặn.
Rule 7: Thông báo phải giải thích lý do được phép hoặc bị từ chối đăng ký.
